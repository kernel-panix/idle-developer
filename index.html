This game auto-saves once a minute, but we offer manual saves for emergencies and testing purposes.

Reset all game data except your All Time stats. Resetting will earn you daemons.

Daemons provide a passive 0.1% bonus to all earnings.

This will delete all current and saved game data. You have been warned.

This is a simple skeleton for an idle game. Everything within this game works more-or-less asynchronously.

Current Features
  • All timers work asynchronously.
  • Auto-saves every minute and auto-loads utilizing localStorage. (With manual save option.)
  • Over 200 achievements and 25 buyable upgrades and growing.
  • Soft resets with software daemon profit boosters for every 4 billion or so you earn, but becomes progressively harder.
  • Display numbers properly formatted using toLocaleString() enabling localization options for the future.
  • Bonuses for buying multiples of things. For example, buying 10 Smallest will double its speed and again when you have at least 25.
  • Manual click mining with automation option.
  • Manual hard reset.
  • Auto-gains for a "manager" power-up.
  • Manual saving functionality and saves on item buying.
  • Proper buyable powerups.
  • The ability to buy in multiples.
  • Tabbed layout.
  • Payout per cycle display.
  • Separation of worker buy buttons and current price.
  • SoundCloud API integration
